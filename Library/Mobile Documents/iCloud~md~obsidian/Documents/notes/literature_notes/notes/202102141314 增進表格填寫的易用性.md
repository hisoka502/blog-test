# 增進表格填寫的易用性
使用 fieldset tag 包覆像是 radio 之類的複數選項，同時可以加入 legend 描述這些選項的主旨標題（例如：性別）。範例：

<fieldset>
	<legend>性別</legend>
	<input id="male" type="radio">
	<label for="male">男</label>
	<input id="female" type="radio">
	<label for="female">女</label>
	</fieldset>
