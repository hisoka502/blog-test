# 快速為 grid 切出多個 cell 的作法
標準的 grid 欄列設定，需要把每個 cell 的單位都標出來，但如果遇上大數量的欄位，這樣作就不合理。所以可以使用 `repeat` 指令，讓我們只需要輸入需要的欄列數量，以及單位即可。
假設我們想要規劃一個有 50 欄的 grid，每欄 10px，只需要這樣寫：
`grid-template-columns: repeat(50, 10px)`
