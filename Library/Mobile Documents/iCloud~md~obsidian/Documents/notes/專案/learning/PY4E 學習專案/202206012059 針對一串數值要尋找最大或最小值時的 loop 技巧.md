若我們有一串數值，想利用 for loop 來尋找最大或最小值，在我們已知 Python 的邏輯是從頭開始檢視到尾，一個一個比較時，則我們可以利用 None 變數，把第一個數值抓出來，利用 if 判斷式來讓第一個數值一一與後面的數值作比較，若滿足大 / 小的條件，就讓數值取代，藉此找出目標值。以找出最小值為例：

smallest = None
print('Before')
for value = [3,6,2,4,7]
    if smallest = None:
        smallest = value
    else smallest > value
        smallest = value
    print(smallest, value)
print('After', smallest)
